The Robotarium Project

What is the Robotarium?

The Robotarium project's objective is to make an online robotics platform that is freely accessible to anyone. Robotics research currently involves multimillion dollar investment to participate. We believe that the people with the best ideas should win instead of those with the deepest pockets. In order to make that possible we are creating a remote robotics lab where anyone can upload code and test their ideas on real hardware.

How do I get involved?

The Robotarium is now open for business! Currently the Robotarium is in closed alpha. In order to be a tester contact Zak Costello or Magnus Egerstedt.

How is it used?

The Robotarium supplies MATLAB and Python APIs for algorithm deployment. Here's a vanilla consensus algorithm in the MATLAB API 

% Declare number of robots
N = 7;

% Initialize Robotarium
robotarium = Robotarium(N);

% Graph Laplacian for a complete graph
L = ones(N, N) - N * eye(N);

while(true)

    % Get the latest pose data from the robots in format 3 x N
    x = robotarium.getPoses()

    % Calculate the control input for the continuous-time consensus
    % dynamics in 2 dimensions
    dx = kron(eye(2), L) * [x(1, :)' ; x(2, :)'];

    % Restructure data
    dx = [dx(1:N)' ; dx(N+1:(2*N))'];

    % Use a provided diffeomorphism to transform 
    % single-integrator to unicycle dynamics
    dx = diffeomorphism(dx, x);

    % Set the velocities of the robots
    robotarium.setVelocities(dx);

end
